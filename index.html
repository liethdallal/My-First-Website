My Projects

Simon Game

I built the Simon game by combining JavaScript, HTML, and CSS. In the HTML, I structured the game's user interface, creating the game board and layout with div elements. JavaScript was essential for implementing interactivity, adding event listeners for player interactions, managing game state, and dynamically updating the DOM to reflect game progress. CSS played a vital role in styling and visual enhancements, defining the game's appearance, positioning, and animations, ensuring an attractive and user-friendly interface. This project demonstrates my proficiency in web development, showcasing how these three technologies collaborate to create an engaging and visually appealing Simon game.
